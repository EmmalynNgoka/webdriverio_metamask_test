# webdriverio_metamask_test

## Installation

Use npm to install the dependencies

```bash
npm install
```

### Command
#### Run all features
```bash
npm run node_modules/.bin/wdio wdio.conf.js --suite
```
#####  Run a specific feature file
```bash
npm run node_modules/.bin/wdio wdio.conf.js --suite=featurename
```
